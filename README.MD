# Memory card 🎮

_A fun memory matching game where players flip cards to find matching pairs. Test your memory by matching all the images before the numbers of attempts run out._

![Screenshot or Logo](image/Memory%20card.png)

## Getting Started

### Play the Game

[Memory Card Game](http://127.0.0.1:5500/index.html)


### How to Play

1. Click on any card to reveal its image.
2. Click a second card to reveal another image.
3. If the two cards match, they will satay face up.
4. If they don't match, they will flip back after a short delay.
5. Try to complete the game before the number of attempts run out
6. Press the **Reset** button to shuffle the cards and play again.

### Installation

No installation required! Simply clone the repository to your local machine and open the `index.html` file in your favorite browser to start playing.

```bash
git clone https://github.com/Zuhair05/project-wordle-game.git
cd Memory Card
open index.html
```
